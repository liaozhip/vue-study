# vue2.X 组件通信方式大全

## 1. props(自定义属性传值)

- **在父组件中(Parent.vue) 向子组件(Child.vue)传值.**
  
```vue
<template>
  <!-- 父组件向子组件传递了一个name属性, 值是 hi. -->
  <Child name="hi" />
</template>
<script>
export default {}
</script>

<template>
  <!-- 或者使用 :(v-bin), 来动态的传递一个属性值. -->
  <Child :name="msg" />
</template>
<script>
export default {
  data() {
    return {
      msg: 'hello'
    }
  }
}
</script>
```

- **子组件使用(Child.vue)里面使用 props 来接收.**

```vue
<template>
  <div>{{ name }}</div>
</template>
<script>
export default {
  props: ['name']
}
</script>
```

- **关于props的更多高级用法:** [vue官网](https://cn.vuejs.org/v2/guide/components-props.html#Prop-%E9%AA%8C%E8%AF%81)

## 2. 自定义事件

- **父组件(Parent.vue)使用 v-on (@) 在子组件(Child.vue)上面定义一个事件名.**

```vue
<template>
  <!-- 在父组件中, 在子组件上面定义 onMessage 事件名, 用于监听父组件的 onMsg 这个函数 -->
  <Child @onMessage="onMsg" />
</template>
<script>
export default {
  methods: {
    onMsg(params) {
      console.log('子组件触发', params)
    }
  }
}
</script>
```

- **子组件通过 this.$emit('事件名', 要传的参数) 来触发这个事件.**

```vue
<template>
  <button @click="onClick('params')">点击触发父组件的onMsg</button>
</template>
<script>
export default {
  methods: {
    onClick(params) {
      this.$emit('onMessage', params)
    }
  }
}
</script>
```

## 3. 事件总线

- **vue提供的全局api: \$on 用来全局监听一个自定义事件, 使用 \$emit 来触发事件, 是一个跨组件通讯的方式.**
  
```javascript
// 在 main.js 中
import Vue from 'Vue';

Vue.prototype.$bus = new Vue();

// 父组件中
this.$bus.$on('eventName', funcName)

// 子组件中
this.$bus.$emit('eventName', args)

// 注意：使用完应该在组件销毁时卸载，否则会造成多次监听同一个事件，触发事件多次执行
```

## 4. provide / inject

- **在父组件(Parent.vue)中使用 provide 来提供数据, 在后代组件中使用 inject 来获取数据.**

```vue
<script>
// 官方提示：provide 和 inject 绑定并不是可响应的。这是刻意为之的。然而，如果你传入了一个可监听的对象，那么其对象的 property 还是可响应的。
export default {
  provide() {
    return {
      message: this.msg
    }
  },
  data() {
    return {
      msg: 'hello children'
    }
  }
}
</script>

<script>
// 在后代组件件中, 用 inject 接收
export default {
  inject: ['message']
}
</script>
```

- **provide/inject 更多高级用法：**[vue官网](https://cn.vuejs.org/v2/api/#provide-inject)

## 5. $root

- **可以访问到根组件的实例. 访问this.\$root.$el相当于访问** `<div id="app"></div>`

```vue
<script>
export default {
  mounted() {
    console.log(this.$root) // Vue {_uid: 1, _isVue: true, $options: {…}, _renderProxy: Proxy, _self: Vue, …}
  }
}
</script>
```

## 6. $parent

- **可以访问到父组件实例.**

```vue
<!-- 父组件(Parent.vue) -->
<template>
  <Child />
</template>

<!-- 子组件(Child.vue) -->
<script>
export default {
  mounted() {
    console.log(this.$parent) // 拿到Parent.vue组件实例
  }
}
</script>
```

## 7. $children

- **可以访问到所有的子组件实例. 注意: 子组件渲染时不保证顺序.**

```vue
<template>
  <Child1 />
  <Child2 />
  <Child3 />
</template>
<script>
export default {
  mounted() {
    console.log(this.$children) // 是一个数据，包含(child1, child2, child3)
  }
}
</script>
```

## 8. $refs

- **用来访问指定的组件实例, 如果不是组件则拿到的是 dom 元素.**

```vue
<template>
  <div ref="dom"></div>
</template>
<script>
export default {
  mounted() {
    // 在组件挂载后可以获取到, 拿到的是一个dom元素
    console.log(this.$refs['dom'])
  }
}
</script>

<template>
  <Child ref="child" />
</template>
<script>
export default {
  mounted() {
    // 在组件挂载后可以获取到, 拿到的是一个Child组件实例
    console.log(this.$refs['child'])
  }
}
</script>
```

## 9. $attrs

- **父组件传递到子组件中的属性如果没有用 props 接收, 在子组件中就可以使用 $attrs 获取到.**

```vue
<!-- 在父组件(Parent.vue)中 -->
<template>
  <Child name="jerry" />
</template>

<!-- 在子组件(Child.vue)中 -->
<template>
  <div>
    <!-- 没有用props接收 name, 就会保存在 $attrs 里面 -->
    {{ $attrs.name }}
  </div>
</template>
```

- **注意: 写在组件上的属性如果没有用 props 处理, 默认会被当成原始属性挂载在子元素节点上, 可以通过在子组件中添加 inheritAttrs: false 来去掉这些默认行为.**

```vue
<template>
  <!-- 如果父组件传进来的 name 没有用 props 处理就会默认挂载到子元素div上面 -->
  <div name="jerry"></div>
</template>
<script>
export default {
  // 可以在这里设置 inheritAttrs 来去掉div上面的属性 name="jerry"
  inheritAttrs: false
}
</script>
```

- **参考:** [vue官网](https://cn.vuejs.org/v2/api/#inheritAttrs)

## 10. $listeners

- **包含了父作用域中的(不含 .native 修饰器的) v-on 事件监听器. 它可以通过 v-on="$listeners" 传入内部组件 —— 在创建更高层次的组件时非常有用.**

```vue
<!-- 父组件中 -->
<template>
  <Child @onMsg="onMsg" @onSubmit="onSubmit" />
</template>
<script>
export default {
  methods: {
    onMsg() {
      console.log('onMsg')
    },
    onSubmit() {
      console.log('onSubmit')
    },
  }
}
</script>

<!-- 子组件中 -->
<script>
export default {
  created() {
    console.log(this.$listeners) // 拿到onMsg和onSubmit数组
  }
}
</script>
```

- **参考：**[vue官网](https://cn.vuejs.org/v2/api/#vm-listeners)
